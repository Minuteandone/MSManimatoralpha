# MSM Animator
MSM animator is a work in progress easy to use animation software specifically designed for my singing monsters projects on scratch. This isn't a release build, so it might not be stable and might be missing features. Please report any bugs you find.

IMPORTANT NOTE: most or all of this program was generated by ai, however all assets (including non-ui text) were made by me. I am not cleaning ownership to any of the code included

## Current prototype
This repo now includes a simple HTML animation maker inspired by Moho:

- Scratch-style stage coordinates (center origin at 0,0).
- Drag-and-pose puppet on a stage.
- Keyframe timeline with editable transform values.
- Play/stop preview with configurable FPS and duration.
- Browser runtime now adapts key casting/math helpers from `Rendering/util/cast.js` and `Rendering/util/math-util.js` for better behavior parity.

### Run locally
You can open `index.html` directly in a browser, or run a static server:

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173>.
